# AI Student Fitness & Diet Planner

A personalized, budget-friendly fitness and nutrition solution designed specifically for students. This application uses AI to generate workout routines and meal plans based on a student's goals, budget, and cuisine.

## Key Features
- **Smart Workouts:** Exercises based on your available equipment.
- **Student-Budget Meals:** Diet plans that won't break the bank.
- **Indian & Global Cuisine:** Tailored for local food preferences.
- **AI-Powered:** Uses Llama-3-8B via Hugging Face.

## Tech Stack
- **Frontend:** Streamlit
- **Backend:** Python
- **AI Engine:** Hugging Face Inference API
- **Deployment:** Streamlit Cloud

## Project Structure
- `app.py`: Main application code.
- `requirements.txt`: Library dependencies.
- `.gitignore`: Security shield for API keys.
- `README.md`: Project documentation.
